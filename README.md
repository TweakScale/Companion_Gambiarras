## Description

**TweakScale Companion Gambiarras** adds (up to date and last minute) workarounds, kludges or plain *gambiarras*  for 3rd parties patchers that are screwing with TweakScale™ or any of the Companions™.

The rationale is, unfortunately, straightforward: some 3rd parties are terribly dismissive about the most basic notions about team work, collaborative efforts or/and minimal quality control (what to say assurance) and it's a sad fact that TweakScale™ ends up in the receiving end by being slandered by less then skilled people when some less than correct patches ends up affecting TweakScale in a less then ideal way...

So the only option left is going nuclear about the problem, brute forcing my way into a saner KSP instalment (while trying hard not to make things worse, always a possibility on the long run).

Currently mangling with:

* WiP

And more to come as the need arises.
